# SQUARE MATCH:
- A small game that uses serial communication between arduino and python. The goal of the game is to match your square with a randomly generated square while under the pressure of a gradually decreasing time and leniency window. 

# TECHNICAL FEATURES:
- State management
- Continuous serial communication
- Real-time interactive gameplay
- Manually calculated geometry
- Adaptive boundary mapping based on scale and rotation
- Scaling difficulty
- Score tracking via file
